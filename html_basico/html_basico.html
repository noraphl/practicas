Look up to the skies and see

I need no sympathy

Because I'm easy come, easy go

Little high, little low
Any way the wind blows

String

Es un tipo de dato que almacena caracteres
de texto. Deben estar encerrados entre comillas.

Array

Es un conjunto de datos (incluso otros
arrays) que se almacena de forma ordenada.
Se define entre corchetes.

Symbol

Es una etiqueta que representa nombres o
cadenas, que referencia a un ID interno,
y que permanece constante en toda la ejecución
del programa.
